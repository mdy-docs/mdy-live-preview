# MDY Live Preview

[MDY Live Preview](https://mdy-preview.com/) is a tiny web tool to preview MDY formatted text.

## Setup

```
$ npm install
```

## Local Development

```
$ npm run dev
```

## Build

```
$ npm run build
```

## Build and Serve

```
$ npm run build-and-serve
```
